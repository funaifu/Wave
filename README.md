# Wave波纹动画<br>
<br>
To get a Git project into your build:<br>
<br>
## Step 1. Add the JitPack repository to your build file<br>
<br>
allprojects {<br>
repositories {<br>
maven { <br>
     {<br>
    url 'https://jitpack.io' }<br>
     }}<br>
<br>    
## Step 2. Add the dependency<br>
<br>
dependencies{<br>
	        compile 'com.github.funaifu:Wave:1.0'<br>
    }
